
# Aplicaciones basadas en tecnologías web - Actividad 1

Este proyecto corresponde a la **Actividad 1** de la asignatura **Aplicaciones basadas en tecnologías web**.

## Descripción

La práctica consiste en desarrollar una **página web estática tipo portfolio personal** utilizando **HTML5** y **CSS3**, tomando como referencia el diseño proporcionado en el enunciado de la actividad.

En esta web se presenta un perfil profesional básico con:

- imagen de perfil
- nombre y presentación profesional
- breve biografía
- enlaces a redes sociales
- botones finales con accesos externos

## Objetivo de la actividad

El objetivo principal de la actividad es maquetar y estilizar una web sencilla de presentación personal, aplicando correctamente:

- estructura con HTML5
- estilos visuales con CSS3
- alineación de elementos
- enlaces externos
- diseño adaptable a distintos tamaños de pantalla

## Tecnologías utilizadas

- HTML5
- CSS3

## Funcionalidades implementadas

- Página principal tipo portfolio
- Imagen de perfil personalizada
- Nombre completo y descripción profesional
- Biografía breve
- Iconos con enlaces a distintas redes sociales
- Botones con enlaces externos
- Diseño simple, limpio y centrado
