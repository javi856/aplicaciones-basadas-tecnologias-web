# Aplicaciones basadas en tecnologías web

Repositorio con prácticas de la asignatura **Aplicaciones basadas en tecnologías web**.

Página web estática de presentación personal desarrollada con **HTML5** y **CSS3**.

## Tecnologías utilizadas

- CSS3
- JavaScript
- Bootstrap
