# JavaScript - Lista de ejercicios

Este directorio contiene una colección de ejercicios prácticos de **JavaScript** desarrollados para la asignatura **Aplicaciones basadas en tecnologías web**.

## Descripción

El proyecto está organizado a partir de una página principal `index.html` que actúa como menú de navegación y permite acceder a los distintos ejercicios realizados.

Cada ejercicio está guardado en su propia carpeta, lo que facilita la organización del código y la separación de funcionalidades.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- Bootstrap
